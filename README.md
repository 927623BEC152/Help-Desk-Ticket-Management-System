# 🎫 Help Desk Ticket Management System using ServiceNow

A cloud-based Help Desk Ticket Management System developed using **ServiceNow** to automate IT support requests, ticket assignment, workflow management, and reporting.

---

## 📌 Project Overview

The Help Desk Ticket Management System simplifies the process of handling IT support requests within an organization. Employees can create support tickets, support agents can resolve them efficiently, and managers can monitor ticket activities through reports and dashboards.

The application automates ticket assignment, email notifications, and ticket lifecycle management, improving overall operational efficiency.

---

## 🚀 Features

- Create Support Tickets
- Automatic Ticket Assignment using Business Rules
- Role-Based Access Control (Employee, Support Agent, Manager)
- Ticket Status Tracking
- Flow Designer Automation
- Critical Ticket Email Notifications
- Reports & Analytics
- Dashboard Visualization
- Secure User Management

---

## 🛠️ Technologies Used

- **Platform:** ServiceNow
- **Automation:** Flow Designer
- **Business Logic:** Business Rules
- **Security:** Roles & ACLs
- **Reporting:** ServiceNow Reports
- **Dashboard:** ServiceNow Dashboard
- **Notifications:** Email Notifications
